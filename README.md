# Simple PHP Application

This is a simple php application with unit test.

# Contents

 - PHPUnit
 - php example
 
# Prerequisites

 - PHP V 7.2
 - Composer
 
# Installation

     git clone git@github.com:GuillaumeJUST/simple-application.git
     cd simple-application/
     composer install
 
# Running 
 
    php index.php

# Test

    composer test
   
With coverage

    composer coverage
